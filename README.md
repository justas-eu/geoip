geoip
===========

Java EE 6 web application.
The application tells ip address geo location.

Technologies
===========
- Java EE 6
- CDI
- JPA

Prerequisites
===========
- Java 6 or 7
- Maven 3
- TomEE or GlassFish 
- In use with http://dev.maxmind.com/geoip/legacy/geolite/

Install
===========
<pre>
mvn install
</pre>

Adding DB resource with id JNDI/localmysql on TomEE or GlassFish
